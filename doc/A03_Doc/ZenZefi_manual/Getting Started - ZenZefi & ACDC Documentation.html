    Getting started

    Preamble

    ZenZefi (Zentrale Zertifikatesverwaltung) is an INTERNAL tool for Mercedes-Benz employees and their business partners. If the tool is shared with external parties (e.g. business partner, supplier) the related legal entity MUST confirm the General Terms and Conditions (GTCs) for ZenZefi.

    Installation

    Current Version

    The current version is SPRINT-25.09.

    IT-Shop

    As a Mercedes-Benz internal user you can download ZenZefi via IT-Shop Mercedes-Benz ZenZefi 1.121.1.0 MUI (20082711) or search for "ZenZefi".

    Please Note

    The IT-Shop package says that it's still the old version 1.121.1.0. After installation and a restart of your machine, you will be updated to the latest version SPRINT-25.09. We are currently preparing an updated IT Shop package which will directly install the latest version.

    If ZenZefi is not available on your local IT-Shop, please ask your local IT to import ZenZefi from the global IT-Shop.

    Don't do a manual and IT-Shop installation simultaneously on your PC, this will lead to unexpected behavior.

    Manual Installation

    We offer ZenZefi for different platforms. Please choose the one according to your needs:

    OS Download Description
    Windows 64bit (standalone) zenzefi.exe
    Windows 64bit (installer) zenzefi.msi Installer version which also installs a service which will
    auto start ZenZefi on boot. If you are upgrading from an older version, make sure to uninstall to old version first.
    Linux amd64 (standalone) zenzefi
    Linux aarch64 (standalone) not available Please use platform independent jar instead.
    Jar zenzefi.jar Platform independent

    Subscribe for Updates

    If you want to get informed when a new version of ZenZefi is released, you can subscribe on the JFrog Artifactory:

    Go to the Artifactory, right click on ZenZefi and click on "Follow". You'll get notified by mail if new version is uploaded.

    Proxy Settings

    As an external user of ZenZefi you might need to configure your proxy to talk to our servers. Just open the ZenZefi UI and go to the Settings tab. If you don't know which proxy server you need to enter, please ask your local IT support.

    Certificate Chain Exchange Series to Testing

    The latest package of ChangeOfCertChain Series to Testing is uploaded to the artifactory.

    If you want to get notified when a new package is uploaded, you can subscribe here. Right click on the directory ChangeOfCertChain and click on follow, as already described here for ZenZefi Updates. You'll get notified via mail once a new package is uploaded.

    Security Recommendations

    • Please ensure that the latest updates for your system are installed.

    • Please confirm that the browser you are currently using to run the application is updated to the latest version.

    • Ensure that the latest version of ZenZefi is always used once it is released.
